## 一、本地要存什么(清单)

单文件 redb `data_dir/growbox.redb` 存**结构化数据**(ACID、write-through、重启不丢);**大二进制**(模型权重)单独放 data_dir 文件,redb 只记指针/版本。

| 数据 | 存哪 | 键 / 形态 | 状态 |
|---|---|---|---|
| 时间线节点 | redb `nodes` | id → {content, role, created_at, embedding, embed_model_ver, hits, stain} | 已有(待加 embed_model_ver) |
| 结论(经验/知识/理解) | redb `conclusions` | id → Conclusion | 已有 |
| 设置 / 项目 | redb `kv` | 字符串键 → JSON | 已有 |
| **指针边** | redb `edges`(待建) | `source\0target` → {topic, heat} | 现为内存 + kv blob,**待磁盘化** |
| **碎片(债)** | redb `fragments`(待建) | 区间 → 未扫标记 | 待做(阶段4) |
| **二级锚点** | redb `anchors`(待建) | node → 锚位置 | 待做(阶段4) |
| **嵌入模型权重** | data_dir 文件(非 redb) | `multilingual-e5-small/*.safetensors`+tokenizer | 待做,首次运行下载 |
| **嵌入模型版本标记** | redb `kv` | `embed_model_ver` | 待做,变了就重嵌 |
| **疲劳/缓存元指标** | redb `kv`(周期落) | 命中率/淘汰数/碎片占比 | 待做(阶段5) |

## 二、存储标准(原则)

1. **disk 是真相,RAM 是工作集**。任何缓存都能从盘重建;内存占用 = 工作集大小,与总记忆量解耦(持久 agent 记忆无界)。
2. **按局部访问键设计**。边按 source 键(一次读一个邻域)、节点按 id 键。**禁止"全量 load 进 RAM"作为访问模式**(现 `Timeline` 全量 load 是要改的)。
3. **大二进制不进 redb**。模型权重等放 data_dir 文件,redb 只存路径 + 版本。
4. **版本化 + 迁移**。schema/模型带版本号;不符则迁移或重算(如换 embedding 模型 → 全量重嵌,维度/空间变了不能半量混用)。
5. **隐私:记忆默认不出本机**。本地 embedding 默认即为此;远程槽是 opt-in,且只把"待嵌文本"发给嵌入服务,不发别的。
6. **write-through 即时落库**,关键状态(节点/结论/边/染色)写即持久,崩溃不丢已学到的。

## 三、待你拍的标准问题
1. **遗忘 / 保留策略**:原始时间线节点永久留,还是冷节点归档/压缩?(飞轮把经验压成知识,但原文是否永存?无界增长下迟早要冷归档。)
2. **导出 / 备份**:用户能不能一键导出/备份自己的记忆库?(是用户的数据,倾向"能"。)
3. **静态加密**:个人记忆落盘要不要可选加密?
4. **data_dir 位置**:跟随 OS 惯例(macOS `~/Library/Application Support/GrowBox`)即可?

## 四、总路线(剩余,按建议顺序)
1. **Embedding**(下一步):脚手架+版本重嵌 → 远程 OpenAI 兼容实现 → candle 本地 e5-small → UI 嵌入槽 → 真机验证同义召回。
2. **指针磁盘化 + 三级缓存**(精确层阶段1.5+2,合并):边落 redb 按 source 键,从当前位置导航,RAM 工作集缓存(1:2:4)。
3. **染色补全**(全量扫跳 Deep / 重查 Light)。
4. **二级索引 + 碎片(债)+ 强制跳转指针**(历史引用,位置键)。
5. **做梦 / 睡眠 / 疲劳 + 潜意识 LLM 仲裁器**(地基三锤之二在此落地;做梦/演练也要检索,见 `补遗/做梦睡眠期也在检索.md`)。
6. **健康监控接真数据**(疲劳/压力/缓存仪表从真指标来,non-假0)。
7. **v1 面板接厚**(DreamPanel/MemoryViz/侧栏仪表等空桩接真后端)。
8. **时间线磁盘后端**(惰性取 + 热尾缓存;牵动 RAG 暴力余弦 → 配 ANN,大工程,可最后)。

## 五、验收标准(怎么算"做好了")
- Embedding:同义不同词("汽车"↔"轿车")能互相召回;切模型自动重嵌;断网时本地仍可用。
- 指针磁盘化:百万级节点下,单次检索内存占用恒定(只 page-in 走过的邻域),不随总量涨。
- 飞轮:多次同类操作后,conclusions 出现压缩率>0 的知识级结论(已有最小闭环)。
- 整体:`cargo test --workspace` 持续全绿;每个子系统带 mock 单测;关键路径有真机验证。
