# 00 总览 + 依赖一览(as-built)

## crate 依赖图(单向)

```
app(growbox-gui)  —— 依赖全部 —— Tauri + 前端(SolidJS) + Agent 循环脊柱 + 执行器注册表 + 潜意识仲裁器(P5 arbiter.rs)
├── learn    飞轮:收集→聚类压缩 + 永久目标调度
├── memory   分层检索(RAG→精确)+ 时间线 + 指针网 + 缓存 + 向量索引 + 上下文组装(P4 context.rs)+ 维护(P5 碎片/做梦/睡眠/疲劳 fragments.rs)
├── safety   沙箱 / 路径分级 / 三种授权
├── llm      LLM 通信(流式/reasoning/工具)+ 嵌入(Embedder)
└── core     共享类型:结论模型 + 执行器 trait(零第三方逻辑依赖)
```

架构公理:**一切能力皆"执行器",Agent 循环是唯一脊柱**(一个注册表、一条分发路径、一处安全门)。
