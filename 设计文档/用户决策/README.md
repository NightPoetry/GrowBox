# 用户决策(原话存档)

本文件夹专存**用户做出的决定性设计决策**,用**用户的原话逐字记录**。

为什么逐字:用户的措辞里带着意图和约束,二手转述会丢掉关键信息。这些是项目最高优先级的设计依据,等同真理来源。

**规矩(对接手 AI 的硬要求)**:
- 用户每做一个决定性/方向性的设计决策,立刻在 `决策日志.md` 追加一条:**先逐字抄原话**,再附一句"解读/落点"。
- 不改写、不精简原话。解读只是为了可执行,不能替代原话。
- 这里的决策与 `设计/`(精炼原则)、`计划/`(实现计划)冲突时,**以这里的原话为准**,并去更新那些文档。
- 接手 AI 开工前必读 `决策日志.md`。
